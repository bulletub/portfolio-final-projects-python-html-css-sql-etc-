OBJECTIVE 1: Personalized Blog Platform
----------------------------------------
✓ Demo Steps:
  1. Show guest landing page (index.html)
     - Explain users can browse without login
   - Show featured posts, categories, latest posts
   
  2. Register a new user account
     - Navigate to Register page
     - Create account with email and password
     - Show successful registration message
   
  3. Login with credentials
     - Navigate to Login page
     - Enter credentials
     - Show successful login and redirect to personalized home
   
  4. Show personalized user dashboard (home_user.html)
     - Display user's feed
     - Show create post functionality
     - Demonstrate user-specific content
   
  5. Access user profile page
     - Show profile customization (display name, bio, profile picture)
     - Display user's own posts
     - Show user's shared posts
     - Display friends and groups
   
  6. Edit profile
     - Update display name
     - Add/change bio
     - Upload profile picture
     - Save changes and verify updates
